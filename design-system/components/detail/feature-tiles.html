</div>

<p>These can be styled with a modifier to the <code>feature-tiles__item</code>, which will cascade to the <code>title__highlight</code> and pseudo-square (if the grid item has one)</p>
<p>Modifiers are in <code>_feature-tiles.scss</code>, currently: <code>accent-red</code>, <code>accent-pink</code>, <code>accent-maroon</code>, and <code>accent-tef-gold</code>.</p>
<p><strong>NB: Use the modifier on this item, not the tile if you want the accent to cascade correctly to the pseudo-square, you don't need to add to the tile as well :)</strong></p>

    .feature-tiles {
  $accent-colors: (
    'red': $red,
    'pink': $pink,
    'maroon': $maroon,
    'tef-gold': $gold-tef,
  );

  @media (min-width: map_get($breakpoint-stages, 'tablet-wide')) {
    @supports (display: grid) {
      display: grid;
      grid-auto-flow: dense;
      grid-template-columns: repeat(12, 1fr);
      grid-template-rows: repeat(16, 1fr);

      &__item {
        position: relative;
        grid-column: span 4;
        grid-row: span 4;
        width: auto !important; // overrides fallback grid sizing

        &:nth-child(1) {
          grid-column: span 4 / -3;
          grid-row: 1 / span 4;
        }

        &:nth-child(2) {
          grid-row: 3 / span 4;
        }

        &:nth-child(3) {
          grid-row: 5 / span 4;

          &:after {
            position: absolute;
            content: '';
            width: 25%;
            height: 25%;
            top: 0;
            left: 100%;
          }
        }

        &:nth-child(4) {
          grid-row: 9 / span 4;
          grid-column: span 4 / -3;

          &:after {
            position: absolute;
            content: '';
            width: 25%;
            height: 25%;
            top: 0;
            left: 100%;
          }
        }

        &:nth-child(5) {
          grid-row: 11 / span 4;
          grid-column: 3 / span 4;
        }

        &:nth-child(6) {
          grid-row: 13 / span 4;
          grid-column: span 4 / -1;

          &:after {
            position: absolute;
            content: '';
            width: 25%;
            height: 25%;
            top: 0;
            right: 100%;
          }
        }

        &:nth-child(n):after {
          background: var(--accent-color, #{$red});
        }
      }

      .tile__content {
        margin-right: 25%; // bit of a silly specific override, but this makes things line up with the grid/squares more nicely.
      }
    }
  }

  .title__highlight {
    --background: var(--accent-color);
  }

  @each $name, $value in $accent-colors {
    &__item--accent-#{$name} {
      --accent-color: #{$value};
    }
  }

  // --

  &__square {

  }

  // --

  a.tile {
    transition: transform 250ms;

    &:hover {
      transform: scale(1.025);
      z-index: 2;
    }
  }

  &:hover {
    a.tile {
      .tile__figure {
        img {
          filter: grayscale(1);
        }
      }

      &:hover {
        .tile__figure {
          img {
            filter: grayscale(0);
          }
        }
      }
    }
  }
}
