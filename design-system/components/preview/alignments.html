Alignments

Elements can be aligned left and right within the normal flow of paragraph content using the classes align-left and align-right. Where no specific size is selected, they'll default to medium width. This is normally fine.

As these elements are floated, always make sure there is enough copy to flow all the way around the element to 'clear' it.

Sizes

Elements can be given a size to restrict their width. Normally you don't need to, but if you want to make something explicitly larger or smaller than its default width when aligned, add:

  • size-
  • size-
  • size-
